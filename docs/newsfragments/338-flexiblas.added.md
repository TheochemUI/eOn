Add `with_flexiblas` (meson) / `WITH_FLEXIBLAS` (cmake) -- link Eigen via [FlexiBLAS](https://github.com/mpimd-csc/flexiblas) for runtime-pluggable BLAS / LAPACK. Same FlexiBLAS-of-MPI pattern this PR uses for MPItrampoline and the dlopen-based LAMMPS / ARTn / IRA / XTB loaders: link once, swap the backend (OpenBLAS, Intel MKL, BLIS, ATLAS, ARMPL, ...) at run time via the `FLEXIBLAS` env var. Mutually exclusive with the legacy `use_mkl` / `USE_MKL` direct-MKL link (FlexiBLAS routes to MKL via `FLEXIBLAS=INTELMKL` at run time). `subprojects/flexiblas.wrap` and `FetchContent_Declare(flexiblas)` pin upstream v3.5.0 so neither build system can ever fail with "no FlexiBLAS".
